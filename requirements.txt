#    没装时 transformers 自动 fall back 到 torch 实现，能跑但慢 ~30%。

#      pip install -U "git+https://github.com/Dao-AILab/causal-conv1d" --no-build-isolation
#    AutoDL 镜像 PyTorch 2.5.1+cu124 + CUDA 12.4 toolkit 实测可编译；
#    cu130 等不匹配镜像会失败（详见 debug_log.md 问题 13）。
